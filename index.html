C#

The following solution displays homes that are below the average home price within a neighborhood.
A web application developed with ASP.NET allows for a front-end view of homes that have been computed to be below a neighborhood's average home price.
The algorithm that makes the calculations was developed using node.js.
The data is extracted from redfin.com every 5 minutes using a cron job and stored in an Azure Database.
The node.js app is deployed to a raspberry pi that sits on an Nginx web server.
